# Beauty Center Spa - versión 1

Sitio web estático profesional para Beauty Center Spa.

## Archivos principales

- `index.html`: página principal.
- `styles.css`: identidad visual y responsive.
- `script.js`: menú móvil, animaciones y formulario de consulta.
- `brief-instagram.md`: resumen de investigación del Instagram público.
- `assets/`: carpeta única con logo, hero, imágenes generadas, material visual y hojas de contacto.

## Vista local

Servidor iniciado en:

`http://127.0.0.1:8051/`

También se puede abrir `index.html` directamente desde esta carpeta en un navegador.

## Nota de investigación

Instagram permitió revisar 27 publicaciones sin iniciar sesión. Para auditar las 197 publicaciones del perfil completo, se necesita iniciar sesión en el navegador y crear una versión 2.
